RetreatScreen

macOS Launcher

⌘⇧Spaceを押すだけで、iPhoneのホーム画面のようなグリッドランチャーがフルスクリーンで立ち上がります。 常駐するアクセサリアプリとして、集中を切らさず次のアプリやファイルにジャンプできます。

特徴を詳しく見る →

Key Features

瞬発力のあるランチャー体験

RetreatScreenは常駐しつつ、呼んだときだけ全面オーバーレイで現れるランチャーです。 ホットキーで開き、Escまたは同じキーで瞬時に閉じるので、作業のリズムを崩しません。

⌘

ホットキートグル

初期設定の⌘⇧Spaceで表示/非表示をトグル。フォーカスが外れたら自動で閉じ、デスクトップを占有しません。

☰

編集モードのグリッド

長押しまたは「編集」で編集モードへ。ドラッグで並べ替え、＋で追加、削除は×ボタンのみ。あとから名前やアイコン画像も指定できます。

∞

ガラスの別荘ランチャー

ガラスエフェクトをまとったパネルで、美しいホームスクリーンの別荘のように整列。好みの配置で置いたショートカットを、呼びたいときに気持ちよく呼び出せます。

Flow

起動から編集までの流れ

1

⌘⇧Space で呼び出す

全画面ブラーのオーバーレイとともにランチャーを展開。Escまたは同じショートカットで即クローズ。

2

アプリ/ファイルを起動

アイコンをクリックしてアプリ/ファイルを起動。コンテキストメニューからFinderで表示も可能です。

3

編集モードで整える

編集モードでドラッグ並べ替え、＋で追加、削除は×ボタンのみ。あとから名前変更やアイコン画像の指定もできます。

Tech Specs

動作環境と安心設計

最新のmacOS体験に合わせた軽さと安心設計。呼び出す瞬間だけ姿を現し、データはMacの中だけにとどまります。
